Output a given word horizontally and multiple times vertically so that each letter in the horizontal word matches the position of that letter vertically.

Input: The first line in the data set is an integer that represents the number of words that follow. Each word is on a separate line.

Output: The horizontal word is to be in the centre of the output. Each vertical word is to read down from the top. The first vertical word uses the first letter of the horizontal word. The last vertical word uses the last letter of the horizontal word.

All outputs are to have the same left edge vertically.

The output is to be formatted exactly like that for the sample output given below.

Assumptions: All letters are upper case. The maximum word length is 10.

Sample Input:
TEST
SAMPLE
ART

Sample Output:

TEST

T
